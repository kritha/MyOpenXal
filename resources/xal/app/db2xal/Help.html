Db2Xal

Abstract:

This program is to generate XAL XDXF (accelerator file) from the SNS Global database. This application uses the standard XAL GUI framework and users should be able to figure out how to use most of the menu selections.

Introduction Getting Started

Introduction

From top to bottom the application's frame is divided into a menu-bar, a tool bar, and a status bar.

Getting Started

The first thing the application needs to do is to select sequence(s) to query. One can select the Choose Sequence(s) button on the tool bar to lauch a sequence selection screen.


back to top

Sequences

Next, you can select which database you would like to perform the query. This can be done from the DataBase in the top menu bar. There are two choices: Production Database and Developement Database. The default one is the Production Database.



Now you are ready to query the database anf generate the XDXF file by clicking the Create XAL File... button on the tool bar. After it finishes, there will be a message box. back to top
